# 🏢 StartShareOffice | 스타트공유오피스 소개 웹사이트 외주

> 클라이언트를 위한 외주 프로젝트
> 디자인부터 프론트엔드 개발 및 배포까지 전 과정을 단독 진행

🌐 [👉 실제 배포 사이트 보기](https://startshareoffice.netlify.app/)

---

## 🗂️ 주요 기능

### 📍 메인페이지 구성
- 풀페이지를 활용해 대표 정보 소개 

### 🗺️ 네이버 지도 연동
- 오피스 위치를 지도에 표시  
- 사용자 중심의 직관적 시각화 구현

### 🏢 공간 소개 페이지 구성
- 회사 소개 / 입주 안내 / 공용 공간 이미지 포함

### 🧩 배포 및 연동 (Netlify)
- 정적 웹사이트 배포 자동화  
- 최초 환경 설정 및 도메인 연동 직접 수행

---
